# **App Name**: Ritmo Saudável

## Core Features:

- Criação e Rastreamento de Hábitos: Usuários podem definir, personalizar e registrar diariamente seus hábitos saudáveis. Inclui uma interface fácil para marcação de conclusão.
- Criação e Gerenciamento de Grupos: Permitir que usuários criem ou se juntem a grupos para participar de desafios e competições saudáveis com amigos ou colegas.
- Gamificação de Pontos e Recompensas: Sistema de pontos para cada hábito completado, conquistas para sequências de hábitos e um feed visual de progresso pessoal.
- Classificação de Grupos (Leaderboard): Exibir a pontuação e o progresso dos grupos em um ranking dinâmico, incentivando a competição amigável e o suporte mútuo.
- Chat de Grupo Simplificado: Funcionalidade de chat básico dentro dos grupos para comunicação, incentivo e partilha de dicas entre os membros.
- Configuração de Perfil de Usuário: Permitir que os usuários gerenciem seu perfil, vejam seu progresso geral e ajustem as preferências do aplicativo.

## Style Guidelines:

- Esquema de cores claro. A cor primária para vitalidade e crescimento: #7CCC2B (verde médio-saturado). Perfeita para elementos interativos e títulos importantes.
- A cor de fundo para uma sensação de leveza e foco: #F2F7EC (um verde muito claro e quase branco). Isso mantém a interface limpa e relaxante.
- A cor de destaque para conquistas e chamadas para ação: #DDFF1F (amarelo-limão brilhante). Garante que os elementos importantes se destaquem.
- Fonte para títulos: 'Space Grotesk' (sans-serif) para uma estética moderna, tecnológica e impactante, alinhada à gamificação.
- Fonte para o corpo do texto: 'Inter' (sans-serif) para garantir legibilidade, uma sensação neutra e profissional para o rastreamento de hábitos e comunicação.
- Utilizar ícones minimalistas e vetoriais, com estilo linear ou preenchido sutilmente, para representar hábitos e ações, mantendo a clareza e a simplicidade da interface.
- Animações sutis e responsivas, como feedback visual ao completar um hábito ou transições suaves entre as telas, para reforçar a gamificação sem distrair.
